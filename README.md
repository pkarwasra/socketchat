# socketchat
Chat between server and client using Python socket using TCP and UDP

More find about TCP and UDP - https://medium.com/@sinwarprem/osi-and-tcp-ip-models-9f176ecc88ce#.gayh16wbc

Run TCP server
```
python tcpserver.py
```

Run TCP Client
```
python tcpclient.py
```

Run UDP server
```
python server.py
```

Run UDP client
```
python client.py
```
